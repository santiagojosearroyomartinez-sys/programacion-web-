Sistema operativo:
Es el software que administra el computador y permite ejecutar programas. También controla el hardware y los recursos del sistema.

Node.js:
Es un entorno que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear servidores y aplicaciones web.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y las respuestas que recibirán los usuarios.

Puerto:
Es el punto de comunicación por donde el servidor recibe las solicitudes. En este caso se utiliza el puerto 3000.

Logs:
Son registros que muestran la actividad y los eventos del servidor. Ayudan a detectar errores y monitorear el funcionamiento.
