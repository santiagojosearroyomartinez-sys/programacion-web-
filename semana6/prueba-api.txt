Al enviar una petición POST a la ruta /registro con el JSON:

{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor responde:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esto demuestra que el servidor recibió correctamente los datos enviados.
